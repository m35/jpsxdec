Interfaces and classes for simulating the MDEC (Motion Decoder) chip found in Playstation 1.

The {@link jpsxdec.mdec.MDEC} is the main entry point for all the functionality of this package.

There are 3 implementations of the Inverse Discrete Cosine Transform:

  • {@link jpsxdec.mdec.SimpleIDCT}
  • {@link jpsxdec.mdec.StephensIDCT}
  • {@link jpsxdec.mdec.IDCT}

All three implement {@link jpsxdec.mdec.IDCTinterface}.

An overview of the MDEC decoding process can be found in the latest STR_file_format.txt document.
