java -jar jpsxdec.jar <options> <input-file> <output-file>

If no arguments are provided, GUI is shown.

Simple example:
java -jar jpsxdec.jar FF7disc1.bin --index FF7disc1.idx
(Open the FF7disc1.idx file to see what media was found)
java -jar jpsxdec.jar FF7disc1.bin --index FF7disc1.idx -d 5 ff7disc1
(Decode media item 5 from the disc, using "ff7disc1" as the base name of the output files)

-c/--channel # Only decode audio channel #.
