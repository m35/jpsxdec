All the known sector types.

All sectors extend {@link IdentifiedSector}, which in turn implements {@link IIdentifiedSector}.

All streaming video sectors implement {@link IVideoSector}.

Square related sectors

  • {@link SectorChronoXAudio}
  • {@link SectorChronoXVideo}
  • {@link SectorChronoXVideoNull}
  • {@link SectorFF8}
  • {@link SectorFF9}

use {@link ISquareAudioSector} and {SquareAKAOstruct}.

{@link UnidentifiedSector} can be used when a sector is unknown, but an interface expects {@link IdentifiedSector}.
