Cross-platform video/audio player.

You don't need the JMF to play videos.

The player utilizes several threads perform different tasks

  1. Demuxer thread ({@link jpsxdec.util.player.DemuxReader})
  2. Video processor thread ({@link jpsxdec.util.player.VideoProcessor})
  3. Audio processor thread ({@link jpsxdec.util.player.AudioProcessor})
  4. Video player thread ({@link jpsxdec.util.player.VideoPlayer})
  5. Audio player thread ({@link jpsxdec.util.player.AudioPlayer} wraps the {@link javax.sound.sampled.SourceDataLine})

Packets of data are passed between these threads via {@link jpsxdec.util.player.MultiStateBlockingQueue}s.

There is also the external Controller thread ({@link jpsxdec.util.player.PlayController}) that is typically the gui.
