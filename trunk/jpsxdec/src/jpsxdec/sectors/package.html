All the known sector types.

All sectors extend {@link jpsxdec.sectors.IdentifiedSector}, which in turn implements {@link jpsxdec.sectors.IIdentifiedSector}.

All streaming video sectors implement {@link jpsxdec.sectors.IVideoSector}.

Square related sectors

  • {@link jpsxdec.sectors.SectorChronoXVideo}
  • {@link jpsxdec.sectors.SectorChronoXVideoNull}
  • {@link jpsxdec.sectors.SectorFF8}
  • {@link jpsxdec.sectors.SectorFF9}

use {@link jpsxdec.sectors.ISquareAudioSector} and {@link jpsxdec.sectors.SquareAKAOstruct}.

{@link jpsxdec.sectors.UnidentifiedSector} can be used when a sector is unknown, but an interface expects {@link jpsxdec.sectors.IdentifiedSector}.
